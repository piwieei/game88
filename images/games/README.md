## 图片要求

- 推荐尺寸：400px × 300px
- 格式：JPG或PNG
- 文件名应与游戏ID完全匹配（全小写，使用连字符分隔单词）

## 游戏类型分类

游戏按照以下类型进行分类：

- 街机（巨人冲刺、上升、矿车列车、水果派对）
- 动作（西部狙击手）
- 益智（寻找猫咪、脑力挑战、颜色分类、立方体配对、停车场混乱）
- 女生（时尚对决）
- 技巧（开锁大挑战）
